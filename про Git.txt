GitHub — веб-сервис для хостинга IT-проектов и их совместной разработки. Основан в 2008 году, в 2018 году платформу купила компания Microsoft. 
Принцип работы: сервис использует систему контроля версий Git, которая позволяет отслеживать изменения в коде и координировать работу нескольких человек над одним проектом. Каждый раз, когда вносятся изменения в код, система создаёт новую «страницу» в истории проекта, позволяя вернуться к предыдущей версии или сравнить различные варианты реализации. 
Функции
Репозитории — место, где хранится проект, содержит все файлы и историю их изменений. Делятся на публичные и частные:
Публичные — доступны всем пользователям, часто используются для проектов с открытым исходным кодом. 
Частные — доступны только конкретным пользователям или командам разработчиков, в них хранятся частные или коммерческие проекты, где требуется ограничить доступ к исходному коду. 
Ветки (branches) — позволяют разработчикам работать над разными версиями проекта одновременно, не нарушая основной код. Основная ветка (main или master) содержит стабильную, рабочую версию проекта. 
Система Issues — инструмент для отслеживания задач, багов и улучшений. Каждая проблема получает уникальный идентификатор, комментарии и статус. 
Pull Requests — механизм для предложения изменений в основном коде — включает не только сам код, но и описание изменений, возможность их обсуждения и проведения code review. 
GitHub Pages — инструмент для развёртывания статических веб-сайтов напрямую из репозиториев на GitHub. 
Тарифы
GitHub предлагает различные тарифные планы: 
Бесплатный план — включает неограниченное количество публичных репозиториев и ограниченные приватные репозитории с основными функциями.
Pro — включает дополнительные функции для индивидуальных разработчиков.
Team — предлагает инструменты для совместной работы и управления проектами.
Enterprise — индивидуальные условия для больших организаций с расширенными возможностями управления и безопасности.
