Порядок работы с системой контроля версий Git включает несколько этапов: установку и настройку программы, создание репозитория, работу с ветками и отправку изменений. 
Установка и настройка
Установить Git с официального сайта. Способы установки зависят от операционной системы:
Windows — скачать установщик с сайта git-scm.com. 
macOS — установить Git через менеджер пакетов Homebrew. 
Linux — установить через стандартный менеджер пакетов дистрибутива. 
Настроить Git под свои нужды, например, задать имя пользователя и электронную почту. Эти данные будут автоматически добавляться ко всем коммитам. 
Создание репозитория
Инициализировать репозиторий командой git init. Создаётся скрытая папка .git в текущей директории, где хранится метаинформация о репозитории: история коммитов, ветки, конфигурация.
Добавить файлы в репозиторий: например, файл README.md — git add README.md, все файлы в текущей директории — git add ..
Зафиксировать изменения — выполнить коммит: git commit -m «Первый коммит: инициализация проекта». Текст в кавычках — сообщение коммита, которое должно кратко и точно описывать внесённые изменения.

Работа с ветками
Создать новую ветку — команда git branch branch_name. После первого коммита Git автоматически создаёт первую ветку, обычно в ней хранят стабильную версию проекта.
Переключиться между ветками — команда git checkout branch_name. Также можно одной командой создать новую ветку и сразу в неё перейти: git checkout -b branch_name.
Слить изменения из одной ветки в другую — команда git merge. Завершается появлением общего коммита для объединённых веток.

Отправка изменений
Отправить изменения в удалённый репозиторий — команда git push. Во время первой загрузки нужно использовать команду с опцией -u — это свяжет локальную и удалённую ветки и синхронизирует их для последующих операций.
Получить изменения из удалённого репозитория — команда git pull. Скачивает изменения из удалённого репозитория в локальный. Если есть конфликты между локальными и удалёнными изменениями, их нужно будет разрешить вручную.
